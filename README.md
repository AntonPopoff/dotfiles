## dotfiles

This repository contains personal collection of dotfiles, managed with [chezmoi](https://github.com/twpayne/chezmoi).

*To use dotfiles without `chezmoi` replace `dot_` with `.` and place them into `$HOME` dir.*

## Configs

#### ~/ (dot_*)
* zsh
* tmux
* IdeaVim

#### ~/.config/ (dot_config/*)
* Alacritty
* Neovim
